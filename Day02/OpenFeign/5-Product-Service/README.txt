REST - Representational State Transfer
It is an architectural style of building web services known as web service.
Web Services is a used when application needs to communicate.
The most common format for interchanging the data - json
Spring also supports REST using web dependency.
To create REST endpoint using Spring, we have to carry out following steps.
1. Create a class
2. Annotate this class with @RestController
3. Add Web dependency
4. Create appropriate methods of rest controller
